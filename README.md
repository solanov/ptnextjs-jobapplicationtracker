# Job Application Tracker

This tracker was made by [PedroTech](https://www.youtube.com/@PedroTechnologies)'s [NextJS 16 Full Course 2026](https://www.youtube.com/watch?v=vCIsrOGNhas). I followed along to learn more about this techstack and other aspects of Web Development.

> As of writing this, I am beginner when it comes to this React framework, Typescript, and Tailwind.

# How to run this in your local repository

```bash
git clone https://github.com/solanov/ptnextjs-jobapplicationtracker.git

npm install

npm run dev
```
